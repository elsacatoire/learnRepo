## Getting Started

### First, run the database


```bash
docker-compose up -d
```


### Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

### See the result

Open [http://localhost:3000](http://localhost:3000) with your browser.
